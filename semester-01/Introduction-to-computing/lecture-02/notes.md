# Lecture 02 – Operating System and Computing Basics

*Topics Covered:*

1. *Operating System (OS) Overview:*  
   An operating system is software that manages all hardware resources of a computer and provides services for computer programs.  
   *Key responsibilities:*  
   - Managing CPU, memory, storage, and devices  
   - Running applications efficiently  
   - Ensuring security and stability of the system  

2. *Types of Operating Systems:*  
   - *Desktop OS:* Windows, macOS, Linux  
   - *Server OS:* Windows Server, Ubuntu Server  
   - *Mobile OS:* Android, iOS  

3. *Functions of an Operating System:*  
   - *Process Management:* Controls running programs and their execution.  
   - *Memory Management:* Allocates and manages memory for applications.  
   - *File System Management:* Organizes, stores, and retrieves files on storage devices.  
   - *Device Management:* Manages input/output devices like keyboard, mouse, and printer.  
   - *Security & Access Control:* Protects system resources and user data.  

4. *System Calls:*  
   - System calls are requests made by a program to the OS to perform tasks that the program cannot do directly, such as reading a file or accessing hardware.  

5. *API (Application Programming Interface):*  
   - APIs allow applications to communicate with the OS or other software without knowing the internal details.  

6. *Other Computing Concepts Introduced:*  
   - *Cloud Computing:* Storing and running applications on remote servers accessed over the internet.  
   - *Virtual Machines (VMs):* Software emulation of a physical computer.  
   - *Containers:* Lightweight isolated environments for running applications.  
   - *Cluster and Grid Computing:* Multiple computers working together to perform complex tasks faster.  

*Example:*  
When you open *Gmail*:  
- The OS handles CPU and memory allocation for the browser.  
- The browser uses system calls to access the network and storage.  
- APIs allow the Gmail web application to interact with the browser and server.  
- Cloud servers process your emails and return the results.  

*Summary:*  
Lecture 02 focused on understanding the operating system, its key functions, system calls, APIs, and how modern computing concepts like VMs, containers, and cloud computing fit into software systems.
