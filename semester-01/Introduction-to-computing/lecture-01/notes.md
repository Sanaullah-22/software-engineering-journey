# Lecture 01 – Introduction to Computing

*Topics Covered:*

1. *What is Computing:*  
   Computing is the process of using computer systems to perform tasks, process data, and solve problems. It includes both *hardware* (physical components) and *software* (programs and applications).

2. *Hardware vs Software:*  
   - *Hardware:* The physical parts of a computer such as CPU, RAM, hard drive, keyboard, and monitor.  
   - *Software:* The programs that run on hardware to perform tasks, e.g., operating systems, applications, and utilities.

3. *Role of Operating System (OS):*  
   The OS manages hardware resources and provides services for applications.  
   Examples of OS: Windows, Linux, macOS.  
   *Function:* Memory management, process scheduling, file management, and device control.

4. *System Interaction Example:*  
   When you open *Gmail*:  
   - Your OS manages CPU, memory, and network resources.  
   - The browser interacts with Gmail servers through the internet.  
   - Multiple layers work together, from hardware to OS to application, to display your email.

5. *Other Concepts Introduced:*  
   - *API (Application Programming Interface):* Allows software to communicate with each other.  
   - *System Call:* A request by a program to the OS to perform a task.  
   - *Cloud Computing:* Running applications and storing data on remote servers.  
   - *Virtual Machines & Containers:* Tools to run software in isolated environments.  
   - *Clusters & Grids:* Systems to handle heavy computing tasks using multiple computers.  

*Summary:*  
The first lecture gave an overview of computing, its components, and how software and hardware interact. Understanding these basics is crucial for building programming skills and software engineering knowledge.
