# Lecture 04 – Programming Languages & Introduction to Python

*Topics Covered:*

1. *What is a Programming Language?*  
   - A programming language is a formal set of instructions that allows humans to communicate with computers to perform specific tasks.  
   - Programming languages translate human logic into instructions a computer can understand.

2. *Binary Language (Machine Language):*  
   - The lowest-level programming language.  
   - Comprised of *0s and 1s*.  
   - Directly executed by the computer’s CPU.  
   - Example: 10110011 01001100

3. *Assembly Language:*  
   - Low-level programming language that uses *mnemonics* instead of binary codes.  
   - Requires an *assembler* to convert it into machine code.  
   - Example: MOV AX, 01h  

4. *Interpreter:*  
   - A program that executes code *line by line*.  
   - Converts high-level language into machine code *at runtime*.  
   - Example: Python interpreter (python script.py)  

5. *Compiler:*  
   - Converts the *entire high-level code* into machine code before execution.  
   - Creates an executable file that can run independently.  
   - Example: C/C++ compiler (gcc program.c -o program)  

6. *Low-Level Language:*  
   - Close to hardware and CPU instructions.  
   - Includes *binary language* and *assembly language*.  
   - Efficient but hard to read and write.

7. *High-Level Language:*  
   - Easier for humans to read and write.  
   - Examples: Python, Java, C++, JavaScript  
   - Needs an interpreter or compiler to run on the machine.

8. *General Purpose Language:*  
   - Can be used to build many types of applications.  
   - Examples: Python, Java, C++

9. *Specific Purpose Language:*  
   - Designed for a specific type of task.  
   - Examples: SQL (databases), HTML (web pages), MATLAB (scientific computing)

10. *Introduction to Python:*  
    - Python is a *high-level, general-purpose, interpreted language*.  
    - Features:  
      - Simple and readable syntax  
      - Supports multiple programming paradigms (procedural, object-oriented, functional)  
      - Large standard library and community support  
    - Example – Hello World:  
      python
      print("Hello, World!")
        
    - Python is widely used for *web development, data science, AI/ML, and automation*.

*Summary:*  
Lecture 04 explained different types of programming languages, their levels, purposes, and how they are executed using interpreters or compilers. Python was introduced as a beginner-friendly, high-level language suitable for a wide range of applications.
